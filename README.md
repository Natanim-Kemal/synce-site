# Synce Guide Site

Official landing and documentation page for **Synce** - A seamless PDF synchronization app for Android and Windows.

## About

This is the guide website for Synce, showcasing features, usage instructions, and download options for the app. Built with modern web technologies to provide users with a clean, informative experience.

## Purpose

- Provide an overview of Synce's features
- Guide users on how to use the app
- Offer download links for Android and Windows platforms
- Serve as the main landing page for synce.natanim.dev
